# 私人工具箱 - 微信小程序

一款为个人打造的实用工具合集小程序，帮你安全管理密码、追踪订阅到期，让生活更有条理。

## 功能特性

### 🔐 密码本
- 安全存储账号密码（加密存储）
- 分类管理（社交/购物/金融/工作/游戏/其他）
- 搜索过滤
- 一键复制账号密码
- 随机密码生成器
- 自定义图标

### 📋 订阅管理
- 记录平台会员到期时间
- 到期倒计时提醒（已过期/即将到期/正常）
- 月费预估统计
- 快捷续期功能
- 常用订阅模板（爱奇艺/Netflix/Spotify 等 16 种）
- 支持月付/季付/半年付/年付

### 🔒 安全保护
- 可选密码锁定（保护密码本访问）
- 修改访问密码
- 数据导出摘要
- 数据清除（二次确认）

### 🧩 可扩展架构
- 模块化设计，新功能只需在 `utils/modules.js` 注册即可
- 已预留备忘录、倒计时等扩展位

## 快速开始

### 准备工作

1. **注册小程序账号**
   - 前往 [微信公众平台](https://mp.weixin.qq.com) 注册小程序
   - 个人开发者只需身份证即可

2. **获取 AppID**
   - 登录后台 → 开发 → 开发设置 → 复制 AppID

3. **安装开发者工具**
   - [下载微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 运行项目

1. 打开微信开发者工具
2. 导入本项目文件夹
3. 在 `project.config.json` 中将 `appid` 替换为你的 AppID
4. 点击编译即可预览

### 分享给朋友

- **体验版**：在开发者工具上传代码后，在后台设置体验版，添加朋友为体验成员
- **正式版**：提交审核通过后，朋友可通过搜索或扫码使用

## 如何扩展新功能

1. 在 `pages/` 下创建新功能页面
2. 在 `app.json` 的 `pages` 数组中注册页面路径
3. 在 `utils/modules.js` 中添加模块配置：

```javascript
{
  id: 'notes',
  name: '备忘录',
  icon: '📝',
  desc: '随时记录灵感想法',
  color: '#52C41A',
  bgColor: '#F0FFF0',
  page: '/pages/notes/notes'
}
```

首页会自动展示新模块入口。

## 数据说明

- 所有数据存储在微信本地 Storage 中
- 密码字段经过 XOR + Base64 加密存储
- 数据不上传任何服务器，完全本地化
- 卸载小程序后数据会丢失，建议定期导出备份

## 技术栈

- 微信小程序原生框架（WXML + WXSS + JS）
- 无第三方依赖，零 npm 包
- 纯本地存储，无后端
